# Diabetic Retinopathy Detection using CNN

> **University Applied AI Coursework — 2023/2024**
> Module: 6COSC020W | Student: Mueed Hussain

A deep learning project that uses a Convolutional Neural Network (CNN) to classify the severity of **Diabetic Retinopathy (DR)** from retinal fundus images. The model classifies images into 5 severity levels and achieved a test accuracy of **~71.6%**.

---

## What is Diabetic Retinopathy?

Diabetic Retinopathy is a diabetes complication that affects the eyes, caused by damage to the blood vessels in the retina. Early detection is critical — if left untreated, it can lead to blindness. This project automates severity classification using deep learning to assist medical professionals.

---

## Severity Classes

The model classifies retinal images into 5 categories:

| Class | Description |
|-------|-------------|
| `NO_DR` | No Diabetic Retinopathy |
| `mild` | Mild DR |
| `moderate` | Moderate DR |
| `severe` | Severe DR |
| `Proliferate_DR` | Proliferative DR (most advanced stage) |

---

## Model Architecture

A custom **Sequential CNN** built with TensorFlow/Keras:

```
Input (224x224x3)
→ Conv2D(32, 3x3) + ReLU + MaxPooling(2x2)
→ Conv2D(32, 3x3) + ReLU + MaxPooling(2x2)
→ Conv2D(64, 3x3) + ReLU + MaxPooling(2x2)
→ Flatten
→ Dense(64) + ReLU + Dropout(0.5)
→ Dense(5) + Softmax
```

- **Optimiser:** Adam
- **Loss:** Categorical Crossentropy
- **Epochs:** 20
- **Batch size:** 32
- **Input size:** 224 × 224 pixels

## Results

| Metric | Score |
|--------|-------|
| Training Accuracy | ~73.9% |
| Validation Accuracy | ~72.7% |
| Test Accuracy | **71.6%** |
| Test Loss | 0.757 |

The model was evaluated against a naive baseline classifier and demonstrated meaningful learning across epochs.

## Dataset

The dataset consists of **3,662 retinal fundus images** split across 5 severity classes. Images were split into:
- **Train:** 2,196 images (60%)
- **Validation:** 733 images (20%)
- **Test:** 733 images (20%)

> **Note:** The image dataset is not included in this repository due to its size. The dataset used is based on the [APTOS 2019 Blindness Detection](https://www.kaggle.com/competitions/aptos2019-blindness-detection) dataset available on Kaggle. Download and place it in the `archive/` directory before running.

## Getting Started

### Prerequisites

```bash
pip install tensorflow==2.15.0 numpy pandas matplotlib scikit-learn opencv-python
```

### Running the Notebook

1. Clone the repository:
   ```bash
   git clone https://github.com/mueedhussain7/diabetic-retinopathy-detection-cnn.git
   cd diabetic-retinopathy-detection-cnn
   ```

2. Download the dataset from [Kaggle](https://www.kaggle.com/competitions/aptos2019-blindness-detection) and place images in `archive/colored_images/` organised by class subfolder.

3. Update the `root_dir` path in the notebook to match your local directory.

4. Open and run the notebook:
   ```bash
   jupyter notebook "AI in DR.ipynb"
   ```

### Using the Pre-trained Model

```python
from tensorflow.keras.models import load_model
import numpy as np
import cv2

model = load_model('diabetic_retinopathy_model.h5')

# Preprocess your image
img = cv2.imread('your_image.jpg')
img = cv2.resize(img, (224, 224))
img = img / 255.0
img = np.expand_dims(img, axis=0)

# Predict
classes = ['mild', 'moderate', 'severe', 'NO_DR', 'Proliferate_DR']
prediction = model.predict(img)
print(f"Predicted class: {classes[np.argmax(prediction)]}")
```
